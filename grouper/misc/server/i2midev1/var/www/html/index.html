Information about accessing the demo server

The Grouper Demo server is for showing the capabilities of various versions of Grouper. It is not for any real or production use. It might go down for maintenance or whatever reason at any time, do not rely on it being up. If you want to use it for a demo, please email grouper-dev to request a timespan where we will try not to do maintenance.
