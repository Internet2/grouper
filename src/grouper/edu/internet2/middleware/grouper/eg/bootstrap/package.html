An API usage guide demonstrating how to bootstrap your Groups Registry by creating a wheel group.

Over the course of the 10 steps below I will incrementally show you how to build an application that can be used to bootstrap your Groups Registry. The steps take during this example will introduce a significant amount of API functionality.

Running ant eg-bootstrap will run each of the example programs in this guide.

  1. Find GrouperSystem subject
  2. Start-and-stop sessions
  3. Find the root stem
  4. Find stem by name
  5. Create stem
  6. Find group by name
  7. Create groups
  8. Find GrouperAll subject
  9. Check for membership in a group
  10. Add a member to a group
