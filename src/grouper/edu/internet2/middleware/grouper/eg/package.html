Grouper API Examples

Unfortunately, there is currently extremely little documentation on how to use the Grouper API. That is something that I am currently working on rectifying.

  • Bootstrapping Your Groups Registry

    This example will walk through you writing an application to bootstrap your Groups Registry. It will demonstrate how to:

    • Find subjects
    • Start-and-stop sessions
    • Find the root stem
    • Find stems by name
    • Create stems
    • Find groups by name
    • Create groups
    • Check for membership in a group
    • Add a member to a group
