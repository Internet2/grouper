h4. Description

Group delete will insert or update a group's uuid, extension, display name, or description (with restrictions)


h4. Features

* If group does not exist, the call will not fail (special result code)
* Lookup group to delete by group lookup (by name or uuid)
* Returns group, can be detailed or not
* Can actAs another user

h4. Group delete Lite service

* Accepts one group to delete
* Documentation: [SOAP|http://viewvc.internet2.edu/viewvc.py/grouper-ws/grouper-ws/doc/api/edu/internet2/middleware/grouper/ws/GrouperServiceLogic.html?root=I2MI&view=co] (click on groupDeleteLite), [REST|http://viewvc.internet2.edu/viewvc.py/grouper-ws/grouper-ws/doc/api/edu/internet2/middleware/grouper/ws/rest/GrouperServiceRest.html?root=I2MI&view=co] (click on groupDeleteLite)
